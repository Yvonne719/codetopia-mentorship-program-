Git states
Working Directory: This is where you're actively editing files.

Staging Area: This is like a waiting room area where you place changes you
intend to commit.

Repository: This is the permanent record of every commit.

Difference between CLI and GUI
A CLI is great when you want to work fast, automate tasks, or handle complex
operations using simple commands.

A GUI is better when you prefer a more
visual and user-friendly experience, especially if you don’t want to
memorize commands.
