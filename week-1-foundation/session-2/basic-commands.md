Initialize a repo: git init (Creates a hidden .git folder in your current directory to start tracking it).

Right commit message format: git commit -m "Fix button alignment on login page" (Use the imperative mood—state what the commit does, not what you did).

Check all commits: git log --oneline (Shows your entire commit history with clean, single-line summaries).

Most used Git command: git status (Your compass—tells you exactly which files are modified, staged, or untracked at any second).

Most used CLI command: cd / ls (Crucial for navigating folders and listing out your files).
